Random Number Approximation of Pi#

In this lab we will use random numbers to estimate $\pi\approx 3.14$.

  1. Letâ€™s look at the library matplotlib! The matplotlib (mpl) library has a sub-library called pyplot (plt). We can access the functions in pyplot by executing the next cell.

import matplotlib.pyplot as plt
import numpy as np
np.random.seed(1232023)

STEP ONE A random number is any number whose decimal is between 0 and 1. Letâ€™s use a random number generator to generate 100 random numbers and then plot them.

N=100
x=np.arange(0,N,1) 
y=np.random.rand(N)
plt.scatter(x, y, s=y, marker='o')

fig=plt.figure(figsize=(1,1))
N=100
x=np.random.rand(N)
y=np.random.rand(N)
plt.scatter(x, y,s=.1,  marker='o')

<matplotlib.collections.PathCollection at 0x7eed82392d30>

Q2. How were random numbers used to generate the plot above?

STEP THREE: The distance d from a point (x,y) to the origin (0,0) is given by the formula

\[d=\sqrt{x^2+y^2}\]

Letâ€™s define a function dist(x,y) to compute this distance.

def dist(x,y):
    d=np.sqrt(x**2+y**2)
    return d

Letâ€™s use our function to check the distance from (3,4) to the origin.

dist(3,4)

5.0

STEP FOUR: Letâ€™s use random numbers to create a plot of points in blue if the distance is less than or equal to 1 and in red if the distance is more than 1.

fig=plt.figure(figsize=(1,1))
for n in np.arange(0,1000,1):
    x=np.random.rand(1)
    y=np.random.rand(1)
    if dist(x,y)<=1:
        plt.scatter(x, y,s=.1,  marker='o', color='blue')
    else:
        plt.scatter(x, y,s=.1,  marker='o', color='red')

Q4. What is the shape of the points in blue?

STEP FIVE: The area \(A\) of a circle with radius \(r\) is \(A=\pi r^2\).

Q5. What is the area of a quarter circle of radius \(r=1\)?

STEP SIX: Note that \(\pi/4\approx .785\). Thus, we would expect about 78.5% of the points in the plot in Step 4 to be in blue. Letâ€™s check this by counting the number of points of each color plotting a total of \(N=100\) points.

fig=plt.figure(figsize=(1,1))
N=100
blue=0
red=0
for n in np.arange(0,N,1):
    x=np.random.rand(1)
    y=np.random.rand(1)
    if dist(x,y)<=1:
        plt.scatter(x, y,s=.1,  marker='o', color='blue')
        blue=blue+1
    else:
        plt.scatter(x, y,s=.1,  marker='o', color='red')
        red=red+1
print("Percentage of Blue Points=", (blue/N)*100)
print("Estimate of Pi=", 4* (blue/N))

Percentage of Blue Points= 80.0
Estimate of Pi= 3.2
