1. Functions#

The Python program language is one of the most popular worldwide. It is used widely by data scientists, a profession that is in high demand and well paid if you are good at it. In this demo, we introduce the concept of a function.

Example

Here we define a function called add(a,b) that computes the sum a+b. Hit shift+enter to execute the next cell which creates our function. The # symbol is followed by a comment about the code

def add(a,b):   #make sure you include the :
    s = a+b   #make sure you indent
    return s  #return is a key word found at the last line of the code defining the function

Here is an an example how we can use the function we just created to compute 5+3. (Always hit shift+enter to execute the cell).

add(5,3)
